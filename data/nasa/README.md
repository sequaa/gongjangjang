# NASA IMS Bearing Data (run-to-failure replay source)

Phase 3 이상탐지/품질 신호 사다리의 **라벨 있는 실데이터 스트림**. 합성 부하 스트림 대신
실제 베어링 run-to-failure 진동 데이터를 per-snapshot scalar 특징 시계열로 전처리해
`SignalSource` nasa 어댑터로 가속 replay 한다.

## 출처 / 라이선스

- **Source URL:** https://phm-datasets.s3.amazonaws.com/NASA/4.+Bearings.zip
  (NASA Prognostics Center of Excellence Data Repository, IMS Bearing dataset)
- **Generated by:** NSF I/UCR Center for Intelligent Maintenance Systems (IMS), with Rexnord Corp.
- **Reference:** Hai Qiu, Jay Lee, Jing Lin. *Wavelet Filter-based Weak Signature Detection
  Method and its Application on Roller Bearing Prognostics.* J. Sound and Vibration 289 (2006) 1066-1090.
- **License:** U.S. Government work — public domain (usa.gov / NASA open data). 재배포·재현 허용, 인증 불필요.

## 검증된 구조 (raw + 동봉 "Readme Document for IMS Bearing Data.pdf" 직접 확인)

다운로드는 중첩 압축이다: `Bearings.zip` → `4. Bearings/IMS.7z` → `1st_test.rar` /
`2nd_test.rar` / `3rd_test.rar` + Readme PDF.

| 항목 | 값 (실측) |
|------|-----------|
| 사용 run | **Set 2** (`2nd_test`) — 단일·깨끗한 고장, run-to-failure |
| 고장 | run 종료 시점 **bearing 1 outer race failure** (그래서 Set 2가 고장 run) |
| 파일 수 | **984** (= 984 스냅샷) |
| 채널 수 | **4** (Set 1만 8채널; Set 2·3은 베어링당 1 가속도계) |
| 채널→베어링 | **Bearing 1 = Ch 1 (column 0)**, Bearing 2 = Ch 2, Bearing 3 = Ch 3, Bearing 4 = Ch 4 |
| 파일 1개 shape | **20,480 rows × 4 cols** (1초 스냅샷 @ 20 kHz) |
| 수집 간격 | 10분/스냅샷 (실 run ≈ 7일: 2004-02-12 10:32:39 → 2004-02-19 06:22:39) |
| 파일명 | 타임스탬프 `YYYY.MM.DD.hh.mm.ss` (사전식 정렬 = 시간순) |

> bearing 1 = column 0 을 추적한다 — outer race 고장이 이 베어링에서 발생하므로 RMS가 run 끝으로
> 갈수록 상승하는 열화 트레이스가 보인다 (baseline RMS ≈0.077 → 후반 ≈0.42, 피크 0.725 @ idx 979).
> RESEARCH의 파일 수/채널 가정값(A1/Open Q1)은 raw·PDF로 검증 완료, 위 값이 정설.

## 산출물

- `features.csv` — 스냅샷당 bearing-1 scalar 특징 시계열 (984행). **커밋됨** (재현 앵커).
  컬럼: `snapshot_index, recorded_at, rms, kurtosis, crest`
  - `rms` = sqrt(mean(x²)),  `kurtosis` = scipy.stats.kurtosis (Fisher/excess),
    `crest` = peak/rms (peak = max(|x|)),  `recorded_at` = 파일명 타임스탬프 → ISO-8601.
- `raw/` — 1GB 원본 (zip/7z/rar/.ascii). **비커밋** (.gitignore).
- `preprocess.py` — raw Set 2 → features.csv 결정적 재생성 스크립트.

## 재생성

raw 가 `data/nasa/raw/2nd_test/` 에 풀려 있어야 한다. numpy + scipy 필요:

```
python3 data/nasa/preprocess.py
```

활성 python3 에 numpy/scipy 휠이 없으면 전처리용 venv 사용 (오프라인 1회):

```
python3 -m venv data/nasa/.venv
data/nasa/.venv/bin/pip install numpy scipy
data/nasa/.venv/bin/python data/nasa/preprocess.py
```

raw 압축 해제 (한 번): `Bearings.zip` unzip → `IMS.7z` (py7zr 또는 7z) →
`2nd_test.rar` (unar/unrar) → `raw/2nd_test/`.
